# reactjs-project-frontend-backend
As a passionate UNISA student and aspiring full-stack developer, I created a ReactJS project that allows users to add, delete, and edit student records seamlessly in a PHPMyAdmin database. I designed the front end with ReactJS to ensure a clean, responsive, and user-friendly interface, while building the back end using Node.js and Express for secure, reliable, and efficient data handling.  This project demonstrates my ability to integrate modern front-end frameworks with robust back-end APIs, ensuring smooth communication with the database. I implemented CRUD (Create, Read, Update, Delete) functionality, validation, and error handling to deliver a practical, real-world solution.  Through this project, I proved my skills in full-stack development, database management, and API integration—showing that I can design not just code, but complete applications that are functional, scalable, and ready for real-world use.
